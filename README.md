Muzily 🎵
Muzily is a web application for music streaming and management. The backend is partially completed, and API routes can be tested via Postman.

🚀 Tech Stack
Backend: Next.js 13, PostgreSQL, TypeScript
Authentication: OAuth
Deployment: Vercel

📌 Setup
Clone the repository:
git clone https://github.com/Rahulcodess/muzily.git && cd muzily

Install dependencies:
npm install

Set up environment variables in a .env file
Run the project:
npm run dev
